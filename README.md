# 📊 E-commerce Sales Analysis (Excel Project)
📌 Project Overview

This project focuses on analyzing e-commerce sales data using Microsoft Excel to extract meaningful insights about business performance. The goal is to understand customer behavior, sales trends, and key factors affecting revenue.

## 🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Understand customer demographics (gender-based insights)
Track weekly sales trends
Detect patterns to improve business decisions
## 🛠 Tools Used
Microsoft Excel
Pivot Tables
Charts & Graphs
Data Cleaning Techniques
Conditional Formatting
## 📂 Dataset Description

The dataset includes the following key fields:

Transaction ID (TX ID) – Unique identifier for each order
Product – Name/category of product sold
Amount – Sales value of the transaction
Week Number – Time-based grouping for trend analysis
Gender – Customer demographic information
## 🔧 Data Cleaning Steps
Removed empty and irrelevant rows
Renamed unclear column headers
Handled missing values (e.g., Unknown gender)
Structured raw data into a usable tabular format
## 📊 Key Analysis Performed
### 1. Sales Analysis
Total revenue generated
Weekly sales trends
High and low-performing weeks
### 2. Product Analysis
Best-selling products
Revenue contribution by each product
### 3. Customer Insights
Gender-based purchasing behavior
Comparison between Male, Female, and Unknown categories
## 📈 Key Insights
Certain products (e.g., clothing/shoes categories) drive higher revenue
Weekly trends show fluctuations, indicating seasonal or demand patterns
Gender-based analysis helps in targeted marketing strategies
